To run 

Frontend : 

           npm install 
           npm run dev
	
Backend : 

          npm start server

admin : 

        npm install 
        npm run dev


To create Docker-image

1. Move to main folder :

       cd Tomato-Food-Delivery-main\Tomato-Food-Delivery-main
3. Run the command :

       docker-compose up --build

eg : 


     Tomato-Food-Delivery-main\Tomato-Food-Delivery-main> docker-compose up --build
